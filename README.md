## Configs:
- API_HASH
  - Get from [Here](https://my.telegram.org).
- API_ID
  - Get from [Here](https://my.telegram.org). 
- ADDR
  - The email bot with imap enabled
- PASSWORD
  - The email bot password 
- ADMIN
  - Your own email as bot admin
